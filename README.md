# Accident_Risk_Predictor

Accident Risk Prediction System built using Machine Learning. This project predicts accident risk based on factors like speed, weather, traffic, and driver behavior, and provides safety suggestions along with report generation.=======

## 📌 About the Project

This project is a simple attempt to understand how machine learning can be used in real-life situations like road safety. The system predicts the chances of an accident based on different driving conditions such as speed, weather, time of day, traffic, and driver behavior.

The idea is to take inputs from the user and give an estimate of how risky the situation is, along with some basic safety suggestions.

---

## 🎯 Purpose

The main purpose of this project is:

* To learn how machine learning works in practice
* To understand how different factors affect accident risk
* To build a small system that can help in decision-making while driving

---

## 🧠 Tools and Technologies

I have used the following tools in this project:

* Python
* Pandas (for handling data)
* NumPy
* Matplotlib (for graphs)
* Scikit-learn (for the machine learning model)

---

## ⚙️ What the System Does

* Takes user input like speed, weather, time, etc.
* Predicts accident risk in percentage
* Shows risk level (Very Low to Very High)
* Gives simple safety suggestions
* Displays a graph for better understanding
* Saves previous results (history)
* Generates a report file
* Allows viewing of the last report

---

## 📊 About the Dataset

The dataset used in this project is not random. It is created based on common-sense real-life driving conditions.

For example:

* Higher speed increases risk
* Driving in fog or rain is more dangerous
* Alcohol increases risk significantly
* Heavy traffic increases chances of accidents
* Beginner drivers are more prone to mistakes

This helps the model learn meaningful patterns.

## ▶️ How to Run the Project

1. Install required libraries:

```bash
pip install pandas numpy matplotlib scikit-learn
```

2. Make sure these files are in the same folder:

* `accident_data.csv`
* Python file

3. Run the program:

```bash
python your_file_name.py
```

4. Use the menu options:

* Predict Risk
* Show History
* Show Graph
* See Previous Report
